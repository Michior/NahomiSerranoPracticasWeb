Ejercicios guía 1

En esta guía fuimos capaz de identificar la estructura básica de un documento web. Tambien aplicamos nuestras primeras reglas de estilo CSS a elementos HTML usando estilos embebidos y hojas de estilo externas. Finalmente aprendimos el uso de los selectores básicos y propiedades de CSS para la utilización de fuentes y estilos.

  1. Ejemplo 2
  2. Estilos Embebidos
  3. Estilos Embebidos con CSS
  4. Paises de Centroamerica
  5. Rápidos y Furiosos

Ejercicios guía 2

Fuimos capaces de utilizar y combinar selectores CCS avanzados, además de conocer la utilidad de las pseudo-clases y pseudo-elementos en CSS. Finalmente creamos páginas web utilizando el Modelo de Cajas, identificando sus respectivas partes (margin, border, padding y content).

  1. Pelicula
  2. Tablero de ajedrez
  3. Mi Biografía

Ejercicios guía 3

En esta guía fuimos capaces de utilizar los diferentes tipos de posicionamiento CSS y diferenciarlos apropiadamente. Además usamos las propiedades CSS relacionadas con la visualización de elementos. Finalmente realizamos maquetaciones de sitios web con diseños fijos y fluidos.

  1. Los Oscares
  2. Ejemplo de menú horizontal
  3. Página principal de viajes (index)
  4. Página de planes de viaje
  5. Página de galeria de viajes
  6. Carta: resultado 1
  7. Carta: resultado 2

Ejercicios guía opcional

En esta guía se puso en practica todo lo aprendido a lo largo de las guías anteriores, tanto en estructura como en diseño de los sitios web.

  4. Ofertas

Ejercicios guía 4

En esta guía creamos y estilizamos formularios, utilizando la diversidad de controles de HTML 5. Aprendimos el uso adecuado de la meta-etiqueta viewport. Tambien definimos correctamente las media queries para aplicar reglas de estilo en función del ancho del navegador. Finalmente, utilizamos los conocimientos adquiridos durante el transcurso de la asignatura para la construcción de las páginas web responsivas.

  2. Ejercicio 2
  3. Ejercicio 3
  4. Formulario de Login

  6. Página de inico de viajes
  7. Página de planes de viaje
  8. Página de galeria de viajes
